# Writing with Clarity

Good prose is like a window pane: the reader looks *through* it at the meaning, not *at* it. Most unclear writing is not a vocabulary problem. It is a structure problem and a courage problem. The method below puts the point where the reader looks for it, then strips the rest.

## The moves

1. **Lead with the conclusion (BLUF / Minto).** State the answer first, then the arguments, then the evidence. Bottom Line Up Front. The reader should be able to stop after the first sentence and still have the point. Build the pyramid down from the apex, not up from the rubble.

2. **Put the point where the reader looks for it.** Readers expect the topic at the start of a sentence and the new, important information at the end. The subject should name what the sentence is about; the verb should carry the action. When a sentence feels murky, you have usually buried the point in the middle.

3. **Apply Orwell's six rules on the revision pass:**
   - Never use a stale metaphor or simile you are used to seeing in print.
   - Never use a long word where a short one does.
   - If it is possible to cut a word, cut it.
   - Never use the passive where you can use the active.
   - Never use jargon if you can think of an everyday equivalent.
   - Break any of these rules sooner than write anything outright barbarous.

4. **Cut, then cut again.** First draft to think; second draft to remove. Most filler hides in qualifiers ("very," "quite," "in order to," "the fact that") and in throat-clearing first sentences. Delete them and the meaning sharpens.

5. **Congruence — don't disguise a guess as a fact.** If you are uncertain, say so plainly: "I think," "the data suggests," "unverified." Confident prose that overstates what you know is not clear; it is misleading. Clarity includes being clear about the *status* of a claim, not only its content.

## The question beneath

Before revising the sentences, ask what the reader actually needs to do or believe after reading. Often the clarity problem is that you have not yet decided what you are trying to say. Decide first; the words follow.

## Compact protocol
```
[ ] Conclusion is in the first sentence/paragraph (BLUF)
[ ] Each paragraph's point is in its first sentence
[ ] Topic at start of sentence, new info at end
[ ] Passive voice replaced with active (where it reads better)
[ ] Long words and jargon swapped for plain ones
[ ] Every removable word removed
[ ] No stale metaphors
[ ] Uncertainty marked honestly; no guess dressed as fact
[ ] A stranger can state the main point in one sentence
```
