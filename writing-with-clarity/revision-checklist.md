# Revision Checklist

Run this over a finished draft. Each line is pass/fail. A fail means revise before you ship.

## Structure
```
[ ] The conclusion appears in the first sentence or first paragraph (BLUF)
[ ] Each paragraph leads with its own point
[ ] Sentences open with the topic and close with the new/important information
[ ] Sections are ordered most-important first, not chronologically by how you thought of them
[ ] A stranger could state the main point in one sentence after reading
```

## Sentences (Orwell pass)
```
[ ] No stale or dying metaphors
[ ] No long word used where a short one fits
[ ] Every removable word has been removed
[ ] Active voice used wherever it reads better than passive
[ ] No jargon that has a plain everyday equivalent
[ ] No sentence kept that is correct but barbarous — broke a rule to fix it
```

## Filler
```
[ ] Qualifiers cut: "very," "quite," "really," "in order to," "the fact that"
[ ] Throat-clearing opener deleted (the real first sentence is the second one)
[ ] No sentence repeats what the previous sentence already said
```

## Congruence
```
[ ] Guesses are labeled as guesses; facts are facts
[ ] Uncertainty stated plainly, not hidden behind confident phrasing
[ ] No claim asserted more strongly than the evidence supports
```

## Final
```
[ ] Read aloud once; every place you stumbled is rewritten
[ ] The window pane test: the reader looks through the prose, not at it
```
